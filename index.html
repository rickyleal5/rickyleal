* Hugging Face Model Serving Platform [GitHub]

Developed a scalable microservices architecture to deploy and serve HuggingFace NLP models. Implemented a Node.js API Gateway (TypeScript) for request routing and load balancing, and utilized FastAPI for efficient model inference. Deployed on AWS EKS with infrastructure managed via Terraform, incorporating API Gateway, ECR, VPC, and Load Balancer Controller to ensure secure and efficient operations.

Technologies: Node.js, TypeScript, AWS EKS, Terraform, Docker, Kubernetes, AWS API Gateway, AWS ECR, AWS VPC, AWS Load Balancer Controller.

* IoT Telemetry Pipeline [GitHub] [Demo Video]

Developed a real-time data pipeline using AWS IoT Core, AWS Lambda, and Timescale Cloud to process telemetry from a Raspberry Pi 3 with an ultrasonic distance sensor. Infrastructure was provisioned using AWS CloudFormation, and data was visualized using Grafana dashboards.

Technologies: AWS IoT Core, Lambda, CloudFormation, Node.js, PostgreSQL, TimescaleDB, Grafana, Raspberry Pi.

* YouTube Trends Dashboard [GitHub] [Demo Video]

Engineered a data pipeline to ingest and analyze trending YouTube video data from Kaggle. Used Node.js and Sequelize to stream data into TimescaleDB and created Grafana dashboards to explore content trends and viewership analytics.

Technologies: Node.js, Sequelize, TimescaleDB, Grafana, PostgreSQL, Kaggle.

* Seabird GPS Data Visualization [GitHub] [Demo Video]

Built a visualization platform for seabird migration data sourced from Dryad. Parsed and processed spatial GPS data using Node.js and Sequelize, stored it in TimescaleDB, and visualized migratory patterns in Grafana.

Technologies: Node.js, Sequelize, TimescaleDB, Grafana, PostgreSQL, GPS data.
